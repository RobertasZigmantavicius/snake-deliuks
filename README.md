# Snake Deliuks

A browser snake game that starts Nokia-style and gradually metamorphoses into something else entirely.

## Play

Open `index.html` with VS Code Live Server — or visit the Vercel deployment.

## Controls

- **Arrow keys** — move
- **Space** — pause tracking / restart after game over
- **Y** — allow sound reactions (unlocks at stage 4)

## Dev

No build step. Open the folder in VS Code, right-click `index.html` → **Open with Live Server**.

## Evolution

6 stages. Early changes are subtle — almost imperceptible. By stage 4 the rules shift. By stage 6 the game knows you're there.

See the design spec for the full arc, trigger system, powerup mechanics, and artifact design.
